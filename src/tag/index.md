## API

| 属性     | 说明       | 类型                                   | 默认值  |
| -------- | ---------- | -------------------------------------- | ------- |
| color    | 颜色       | 'primary' 'success' 'warning' 'danger' | primary |
| size     | 大小       | 'samll' 'large' 'default'              | default |
| text     | Tag 的文本 | string                                 | ""      |
| closable | 关闭标签   | boolean                                | false   |
